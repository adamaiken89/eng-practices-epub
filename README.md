## 1. Preparation

```bash
brew install mdbook // Read on browser
cargo install mdbook-epub // Read on e-reader
```

## 2. Run locally

```bash
mdbook serve
```

## 3. Export to EPUB

```bash
mdbook-epub -s true
```
